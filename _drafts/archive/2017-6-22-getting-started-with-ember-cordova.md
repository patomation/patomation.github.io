#### Installation Steps
1. Install Ember CLI
```
npm install -g ember-cli
```

2. Clone the ember-cordova repo.
```
git clone https://github.com/isleofcode/ember-cordova.git
```

3. Change to ember-cordova directory
```
cd ember-cordova
```

4. NPM install
```
npm install
```


5. Install Ember-Cordova
```
ember install ember-cordova
```
